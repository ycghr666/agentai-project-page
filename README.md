# AgentAI Project Page — Refined Version

This package contains the refined CMU-style static project page for AgentAI.

## Files
- `index.html` — main page
- `styles.css` — styling
- `script.js` — small scroll/reveal behavior
- `assets/` — screenshots and visuals
- `agentai_final_presentation.pptx` — final slides
- `agentai_project_record.pdf` — project record

## Publish with GitHub Pages
1. Create a new public GitHub repository.
2. Upload all files in this folder to the repository root.
3. In GitHub: `Settings` → `Pages`.
4. Set Source to `Deploy from a branch`.
5. Choose branch `main` and folder `/ (root)`.
6. Save and wait for the site URL.

## Before publishing
- Replace the demo placeholder with your final video link or embed.
- Check that the images in `assets/` are all uploaded.
- Update any resource links if you want to add a final report or live demo.
